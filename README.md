Journey Mice & Cheese
(การเดินทางของหนูและชีส)
เกี่ยวกับหนูที่พยายามจะเก็บชีสให้ครบโดยชีสแต่ละชีสมีคะแนน 10 คะแนน
และก็จะมีแมวเลือกให้มันเป็นอุปสรรคหรือสิ่งกีดขวางไม่ให้หนูผ่านไปได้โดยมันจะอยู่ประจำ
ตำแหน่งแต่ละจุดเป็นทั้ง Object นิ่ง และ Object ที่เคลื่อนที่มาใกล้เป้าหมายนั้นก็คือ Cheese
ถ้าหนูจะเก็บชีสต้องหลีกเลี่ยงพวกมันและถ้าเอา Object หนูไปชน Object แมวจะทำให้เสียหัวใจ
1 ดวง, เสียชีสที่เก็บมาได้ด้วยโดยมันจะ respawn ทั้งหมดที่จุดเดิม, เสียคะแนน -10 คะแนน
และถ้าหากเอาหนูไปชนแมวพิเศษก็คือ RandomCat ร่วงลงมาจากข้างบนจะเสียหัวใจ 2 ดวง
และเสียคะแนนด้วยเก็บชีสครบและหากชนแมวทำให้คะแนนไม่เท่ากับจำนวนที่เก็บหรือเก็บครบก็
สามารถผ่านด่านได้ จะมีแมวสีส้มที่ถ้าหากไปหามันได้ก็จะผ่านด่านนั้นไป ในด่าน Level แต่ละ
Level 1 – 5 จะมีการบอกจำนวนชีวิต Scores ที่เก็บได้ และจับเวลาในเกมส์/ด่าน 60 นาที ไล่
ระดับความยากเรื่อยหลักๆ คือเพิ่มจำนวนแมว/แมวพิเศษ RandomCat เร็วขึ้น
